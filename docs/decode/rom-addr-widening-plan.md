# Microcode ROM Address Widening + ECP5 Decoder Measurement — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove the hardcoded 256-slot microcode ceiling by auto-sizing the
decoder's `op.addr` width in the generator (`addr_bits = max(8, bits.Len(numSlots))`),
and measure whether the FPGA synth target should switch from the combinational
`direct` decoder to the BRAM-backed `rom` decoder to reduce J4 ECP5 area.

**Architecture:** The decoder VHDL is generated by `decode/gen-go/` from a TOML
spec; the microcode is a flat slot space addressed by an 8-bit `op.addr` that
advances +1 per slot. We thread a single computed `AddressBits` value through the
model into every place the width is emitted, keeping the 8-bit path byte-identical
(so base/J1/J2 are unchanged) while enabling 9-bit (512 slots) once a spec exceeds
256 slots. Separately, we measure `direct` vs `rom` decoder cost on ECP5.

**Tech Stack:** Go 1.26+ (generator), VHDL-93 (decoder + GHDL sim), yosys+ghdl
plugin (ECP5 synth), `math/bits` (already imported).

**Execution order:** Phase A (measurement) first — it is the actual slowdown
lever and yields immediate data. Phase B (widening) is forward-infra for PM3 and,
because base=251/J4≈255 slots both still fit 8 bits, produces **no change to
current generated output**; it is validated by a synthetic >256-slot fixture.

**Branch:** `decode/rom-addr-widen` (already created off `origin/master`). The
design spec is `docs/decode/rom-addr-widening-design.md`.

```go
package model

import (
	"math/bits"

// addrBitsFor mirrors build.go's formula; if you refactor the formula into a
// helper, call that here instead.
func addrBitsFor(numSlots int) int {
	b := bits.Len(uint(numSlots))
	if b < 8 {
		return 8
	}
	return b
}

func TestAddrBitsBoundaries(t *testing.T) {
	cases := []struct{ slots, want int }{
		{1, 8}, {251, 8}, {255, 8}, {256, 9}, {257, 9}, {511, 9}, {512, 10},
	}
	for _, c := range cases {
		if got := addrBitsFor(c.slots); got != c.want {
			t.Errorf("addrBitsFor(%d) = %d, want %d", c.slots, got, c.want)
		}
		// Sentinel (all-ones) must be a reserved, unused slot.
		if c.slots >= (1 << addrBitsFor(c.slots)) {
			t.Errorf("slots %d leave no sentinel room at %d bits", c.slots, got)
		}
	}
}
```

```go
func romConstBody(words []model.ROMWord) string {
	lines := romLines(words)
	if len(lines) == 0 {
		return "    constant microcode_rom : mem := ();\n"
	}
	var b strings.Builder
	fmt.Fprintf(&b, "    constant microcode_rom : mem := (%s\n", lines[0].Text)

		fmt.Fprintf(&b, "    %s\n", rl.Text)
	}
	return b.String()
}

func romLines(words []model.ROMWord) []ROMLine {
	n := len(words)
	var lines []ROMLine
	i := 0
	for i < n {
		start := i
		for i < n && words[i].Comment == "" {
			i++
		}
		if i < n {
			var b strings.Builder
			for j := start; j <= i; j++ {
				fmt.Fprintf(&b, "%d => \"%s\", ", j, words[j].Bits)
			}
			fmt.Fprintf(&b, "-- %s", words[i].Comment)
			lines = append(lines, ROMLine{Text: b.String()})
			i++
		} else {
			var b strings.Builder
			for j := start; j < n; j++ {
				fmt.Fprintf(&b, "%d => \"%s\"", j, words[j].Bits)
				if j < n-1 {
					b.WriteString(", ")
				}
			}
			b.WriteString(");")
			lines = append(lines, ROMLine{Text: b.String()})
			return lines
		}
	}
	return lines
}
```
